# 🌐 HTML Form Project

This is a simple web application built using **HTML, CSS, Node.js, Express, and EJS**.
It allows users to enter their **name and email**, and displays the submitted data dynamically.

---

## 🚀 Live Demo

👉 Click here to view the project:
🔗 https://html-1-steel.vercel.app/

---

## 📌 Features

* 📝 User input form (Name & Email)
* ⚡ Server-side rendering using EJS
* 🔄 Dynamic data display
* 🌐 Simple and clean UI

---

## 🛠️ Tech Stack

* HTML
* CSS
* Node.js
* Express.js
* EJS


## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/html-1.git
```

2. Navigate to project folder:

```
cd html-1
```

3. Install dependencies:

```
npm install
```

4. Run the server:

```
node server.js
```

5. Open browser:

```
http://localhost:3000
```
